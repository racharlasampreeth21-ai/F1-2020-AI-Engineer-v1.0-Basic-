# F1-2020-AI-Engineer-v1.0-Basic-
Project F1 v1.0 is an AI race engineer for F1 2020 featuring live UDP telemetry, voice commands, AI (Ezio), speech recognition, female voice, multithreading and a live overlay. Currently supports speed, gear, RPM, throttle, brake and DRS. Fuel, tyres, ERS, lap data, strategy and coaching are planned for v2.0. 🏎️
